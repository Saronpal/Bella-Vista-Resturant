## Project Overview

**Bella Vista** is a professional, responsive static website template designed for an authentic Italian restaurant. The design emphasizes a clean, elegant aesthetic with a focus on ease of navigation, a clear menu presentation, and a prominent reservation form.

This project uses standard HTML5 and CSS3, demonstrating strong principles of modern web design, including CSS variables, custom animations, and responsive layouts via media queries.

## üöÄ Features

* **Responsive Design:** Optimized layout for desktop, tablet, and mobile screens using CSS Grid and Flexbox.
* **Fixed Header/Navigation:** Ensures easy access to all sections (`Home`, `Menu`, `About`, `Contact`) regardless of scroll position.
* **Hero Section:** Full-screen background image with a clear call-to-action (View Menu / Make Reservation).
* **Detailed Menu Section:** Clean, organized presentation of food items with prices, easily expandable for additional categories.
* **About Section:** Split layout showcasing the restaurant's story alongside an interior image.
* **Contact & Reservation Form:** Includes detailed contact information and a functional multi-field form for bookings.
* **CSS Variables:** Utilizes `:root` variables for easy theme customization (color palette).
* **Subtle Animations:** Employs CSS keyframes (`fadeInUp`) for elegant entry animations on content sections.
